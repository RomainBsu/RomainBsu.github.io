Root Me

  • Ceci est la page principale du site web créé lors d'un projet au Pays de Galles. Nous étions en équipe de 3 pour réaliser l'entièreté du projet en 2 semaines. Nous avons choisi de respecter la charte graphique du site web en appliquant les couleurs primaires et secondaires du drapeau gallois. Nous avons également réimaginé chacun des onglets dans un style plus moderne. De plus, nous avons mis en place la possibilité de changer la langue en gallois afin de respecter le cahier des charges demandé.

  • La plupart du travail sur cette page se trouve dans le code. En effet, nous avons créé une page de connexion en PHP qui communique avec la base de données pour vérifier l'utilisateur. Nous avons également mis en place une protection contre les injections SQL. Nous aurions également aimé ajouter un filtre qui empêche toute utilisation d'attaques XSS, par exemple en utilisant des CSP ou des filtres côté serveur. Cependant, le temps imparti pour la réalisation du projet ne nous a pas permis de mettre l'accent sur la sécurité, et étant donné une contrainte temporelle, nous avons fait le choix de ne pas l'implémenter.

  • La seconde grande partie de ce projet réside dans le panneau d'administration que j'ai entièrement développé manuellement. Celui-ci permet de contrôler très simplement, après une authentification, les utilisateurs, le calendrier, les messages, et offre la possibilité d'envoyer un e-mail à tous les contributeurs. Il accède directement à la base de données, ce qui permet d'actualiser le site en temps réel avec des actions très accessibles et compréhensibles.

  • Nous avons créé un projet en groupe de deux personnes dans lequel nous avons réalisé un interphone connecté. La page principale présentée ici est le menu principal de notre application.

  • Le projet de web scraper a été créé sur Visual Studio en C#. Il utilise le framework .Net Windows Form. Nous devions réaliser un web scraper sur le site de notre choix et stocker les informations choisies dans une base de données en local.

    L'application recherche toutes les informations sur le site web, les classe et les trie afin de faciliter la navigation pour l'utilisateur. Une fois cela fait, l'utilisateur peut choisir quelles informations stocker dans sa base de données.

  • L'action de scrap est réalisée à l'aide du package "HtmlAgilityPack".
    Le projet a également impliqué l'utilisation de la programmation orientée objet pour le stockage des informations (artistes, albums, etc.), ainsi que pour la connexion à la base de données. J'ai décidé d'utiliser l'outil intégré à Visual Studio utilisant les bases de données SQL pour stocker les informations.
